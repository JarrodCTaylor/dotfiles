# Dotfiles

## Installation

``` bash
git clone https://github.com/Paul1987/dotfiles.git ~/dotfiles
cd ~/dotfiles/install-scripts
bash OSX/setup.sh
# or
bash Linux/install-packages.sh
bash Linux/create-symlinks.sh
```
